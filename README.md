## 🔥 我們解決的問題有多大？

**全球每天上傳超過 500 小時的 YouTube 影片、200 萬支 TikTok、無數 B 站投稿。**

這些影片裡藏著：
- 你競爭對手的策略和話術
- 你所在行業的最新趨勢
- 別人花了幾年摸索出來的經驗
- 可以直接複製的商業模式

**但問題是：影片是最難被搜尋、最難被分析的內容格式。**

你不能 Ctrl+F 搜影片。你不能快速掃過 20 支影片的重點。你只能一支一支看完，然後憑記憶寫筆記。

一支 10 分鐘的影片 = 30 分鐘的分析時間（看完+筆記+整理）。
20 支影片 = **10 個小時**。

**而我們的工具，10 分鐘搞定。**

## 💰 對你的好處

**1. 時間：** 20 支影片從 10 小時 → 10 分鐘（效率提升 60 倍）

**2. 競品透視：** 把對手的影片全部轉成文字，逐字分析他們的話術、產品賣點、CTA 策略。他們花 3 個月拍的內容，你 10 分鐘就全看完了。

**3. 內容再創作：** 影片 → 文字 → 改寫成你自己的文章/貼文/課程。一魚多吃，一個影片變成 5 種內容格式。

**4. 市場研究：** 批量分析一個頻道 100 支影片，找出哪些主題播放量最高、觀眾最愛什麼。用數據決策，不靠直覺。

**5. 語言無障礙：** 日文影片聽不懂？轉成文字後丟翻譯。中文影片要給外國團隊看？轉文字翻譯就好。

## 🧠 技術難點在哪裡？

**「下載影片 + 轉文字」聽起來簡單，實際上超麻煩：**

| 難點 | 說明 | 我們怎麼解決 |
|------|------|-------------|
| **平台限制** | B 站要登入、TikTok 防爬蟲、IG 要 Cookie | yt-dlp 支援 1000+ 平台 + Cookie 自動偵測 |
| **語音辨識** | 中文辨識準確度低、背景音干擾 | Whisper large-v3（OpenAI 最強模型，中文 95%+） |
| **本地運行** | 雲端工具要上傳影片、有隱私風險 | 全部在你電腦上跑，影片不外傳 |
| **批量處理** | 一支一支手動操作效率低 | 批量腳本 + 自動摘要報告 |
| **Cookie 管理** | B 站需要登入但 Cookie 提取很技術 | CDP 一鍵提取，不用裝插件 |

**每一個難點單獨都有解法，但沒有人把它們整合成一條龍。我們做了。**

## 🔗 為什麼要整合？

市面上的工具都是分散的：

```
❌ 傳統做法：
yt-dlp 下載 → 手動轉檔 → 另一個工具轉文字 → 手動貼到 ChatGPT → 手動整理

✅ 我們的做法：
一個指令 → 全自動完成 → 直接拿結果
```

**分散的工具 = 每次都要花 15 分鐘設定環境、處理格式、手動串接。**

**整合的工具 = 30 秒。**

而且整合後才能做到「批量分析」— 你不可能手動對 20 支影片重複做 15 分鐘的設定。但一個批量腳本，3 秒鐘啟動，自動跑完 20 支。

## 🚀 兩種使用方式

### 方式 A：OpenClaw Skill（推薦 — 隨裝即用）

已經有 OpenClaw（龍蝦）？一行安裝，直接對話使用：

```bash
# 安裝
openclaw skills install video-analyzer

# 然後直接跟龍蝦說：
# 「幫我分析這支 YouTube 影片 https://...」
# 「把這個 B 站頻道最新 10 支影片轉成文字」
# 「分析這支影片的重點，整理成 3 個要點」
```

**不需要記指令、不需要打參數。用自然語言就好。** 龍蝦會自動判斷要下載哪支影片、用什麼語言辨識、輸出什麼格式。

### 方式 B：獨立腳本（不需要 OpenClaw）

不想裝龍蝦？也可以直接用腳本：

```bash
# 安裝依賴
brew install yt-dlp ffmpeg
pip install mlx-whisper  # Mac Apple Silicon
# 或 pip install openai-whisper  # Windows/Linux

# 單支影片
./analyze.sh https://www.youtube.com/watch?v=xxx zh

# 批量分析整個頻道
./batch-analyze.sh --channel https://www.youtube.com/@channel --limit 20
```

**兩種方式都免費、都開源、都在你電腦上跑。**

| ✅ IG Reels | 部分需要 | 私人帳號需 Cookie |
| ✅ **B 站** | **需要登入** | 提供 Cookie 即可（見下方教學） |

### B 站 Cookie 設定

B 站限制未登入用戶，所以你需要先登入再匯出 Cookie。

**最簡單的方法：**
```bash
# 如果你用 Comet 瀏覽器（已登入 B 站）
./export-cookies.sh 9333 bilibili
# 自動提取 Cookie，之後 analyze.sh 會自動使用
```

**其他方法：**
- Chrome 已登入 → 安裝「Get cookies.txt」擴充套件匯出
- 把 `cookies.txt` 放在腳本同目錄就好

## 🤔 常見問題

**Q：這合法嗎？**
A：本工具僅供個人學習和研究使用。請遵守各平台的服務條款。

**Q：需要 GPU 嗎？**
A：Mac Apple Silicon 用 `mlx-whisper` 不需要 GPU，速度也很快。Windows/Linux 有 GPU 會更快但不是必要。

**Q：影片很長怎麼辦？**
A：沒問題，Whisper 可以處理任意長度。1 小時影片大約需要 5-10 分鐘轉文字。

**Q：準確度如何？**
A：Whisper large-v3 的中文辨識準確度約 95%+。專有名詞可能會有誤差。
